Це проста веб-сторінка, створена для перевірки роботи веб-сервера.

Про цей сайт

Цей сайт є демонстраційним прикладом статичної HTML-сторінки. Він створений з метою протестувати роботу Apache у контейнері Docker. У майбутньому сюди можна додавати власні сторінки, картинки та інший контент.

Для чого він потрібен?

Основна мета цього сайту – бути простою точкою перевірки. Коли ти відкриваєш http://localhost:8080, то бачиш результат налаштованого веб-сервера. Це дозволяє переконатися, що Docker-контейнер працює коректно.

Що можна додати?

Сюди можна вставляти текстові блоки, зображення, створювати навігаційне меню чи навіть цілі сторінки з інтерактивним контентом. Для простоти ми додамо стильовий файл style.css, який зробить сторінку приємнішою для ока.
